### Scene 2: 소비자 로그인 (30초)
1. 모바일 앱 실행
2. **Consumer** 선택 → 전화번호 `010-2000-0001` 입력
3. 로그인 → "김민수" 대시보드 진입
4. 기존 에스크로 1건 표시 (강남 블루보틀, 1/3 릴리즈됨)

> "소비자 김민수의 대시보드입니다. 이미 1개월치가 카페에 지급되었고,
> 나머지 2개월치는 XRPL 블록체인에 안전하게 보관 중입니다."

### Scene 3: 새 에스크로 생성 (1분)
1. **+** 버튼 탭 → 사업자 선택 화면
2. "파워짐 피트니스" 선택
3. 금액: `300,000` RLUSD, 기간: `3`개월 입력
4. 에스크로 생성 → 월 100,000 × 3회 분할

> "헬스장 3개월 선불 30만원. XRPL Token Escrow가 매월 10만원씩
> 자동으로 분할 보관합니다. XLS-85 — XRPL의 최신 기능입니다."

### Scene 4: 사업자 릴리즈 (1분)
1. 로그아웃 → **Business** 전환
2. `010-1000-0001` (강남 블루보틀) 로그인

> "사업자는 매월 약속된 금액을 청구합니다.
> XRPL이 FinishAfter 시간을 확인하고 자동으로 RLUSD를 전송합니다."

### Scene 5: 소비자 보호 — 취소 (1분)
1. 로그아웃 → Consumer로 다시 로그인
2. 에스크로 상세 → 취소 버튼
3. 남은 pending 엔트리만 환불, 이미 released된 건 유지

> "만약 사업자가 폐업하면? 소비자는 남은 금액을 즉시 환불받습니다.
> 이미 이용한 달은 사업자에게 지급 완료, 미이용분만 돌려받습니다.
> 중재자 없이, 블록체인이 자동으로 처리합니다."

# Vision & Core Values
> Created: 2026-04-26 22:40
> Last Updated: 2026-04-26 22:40

## 1. 핵심 문제 (Problem)

한국에서 헬스장, 학원, 미용실 등의 선불 결제 피해가 매년 반복된다.
사업자가 폐업하면 소비자는 남은 이용권 금액을 돌려받을 방법이 사실상 없다.

- 소비자보호원 통계: 선불식 할인권 관련 피해 상담 연간 수만 건
- 기존 해결책(소비자보호원 중재, 법적 소송)은 느리고 회수율이 낮음
- 소비자는 선불 결제를 기피하게 되고, 성실한 사업자도 신뢰를 얻기 어려움

## 2. 비전 (Vision)

**"선불 결제에 신뢰를 더하다"**

블록체인 에스크로를 통해 소비자의 선불금을 보호하고, 사업자에게는 신뢰 인증을 제공하여
선불 결제 생태계 전체의 신뢰도를 높인다.

## 3. 핵심 가치 (Core Values)

| 가치 | 설명 |
|:---|:---|
| 자동 보호 | 사업자 폐업 시 남은 금액이 XRPL Escrow를 통해 자동 환불 |
| 투명성 | 모든 결제 흐름이 블록체인에 기록되어 누구나 검증 가능 |
| 상호 이익 | 소비자는 환불 보장, 사업자는 "선불금 보호 인증" 마케팅 활용 |
| 단순함 | 블록체인 기술을 몰라도 사용 가능한 직관적 UX |

## 4. 타겟 오디언스 (Target Audience)

### Primary - 소비자
- 한국 내 선불 결제 이용자 (헬스장, 학원, 미용실, 필라테스 등)
- Pain Point: 사업자 폐업 시 환불 불가 불안감
- 현재 해결 방식: 단기 결제 선호, 또는 피해 감수

### Secondary - 사업자
- 선불 결제를 운영하는 소규모 사업자
- Pain Point: 소비자 불신으로 인한 장기 이용권 판매 어려움
- 현재 해결 방식: 할인 프로모션으로 신뢰 대체 시도

### MVP 우선 타겟 업종
1. 헬스장 / 피트니스센터
2. 학원 / 교육기관
3. 미용실 / 네일샵

## 5. 성공 지표 (Success Metrics)

| 지표 | MVP 목표 |
|:---|:---|
| Escrow 생성 | 데모에서 정상 생성 및 월별 릴리스 동작 |
| 자동 환불 | 사업자 폐업 시나리오에서 남은 금액 환불 성공 |
| 데모 시연 | 소비자 결제 -> 월별 정산 -> 폐업 환불 전체 플로우 3분 내 시연 |
